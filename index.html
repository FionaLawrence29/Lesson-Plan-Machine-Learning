Lesson Plan: Introduction to Machine Learning

1. Target Audience

BCA Students (High Graders)

2. Duration

2–3 hours (can be split into two sessions)

3. Learning Objectives (SMART)

  • Define machine learning and explain its significance within 15 minutes of instruction.
  • Differentiate between supervised and unsupervised learning with 80% accuracy.
  • Describe real-world ML applications in a group activity within 20 minutes.
  • Classify problems into appropriate ML categories with 80% accuracy in a quiz.

4. Materials Required

  • Projector or smart board
  • Slide deck, handouts
  • Internet-enabled devices
  • Multimedia resources (YouTube, ML tools)

5. Teaching Methodology

  • Mini-lecture, discussion, Think-Pair-Share
  • Group activities, multimedia integration

6. Content Outline

  • Hook: ML video + Mentimeter poll
  • Intro to AI/ML + Types
  • ML Workflow & Tools (Google Teachable Machine)
  • Applications and real-world examples

7. Classroom Activities

  • "You are the Algorithm" role play
  • Classification game
  • Jamboard brainstorm
  • ML Pictionary/Kahoot

8. Assessment

  • 10 MCQs
  • 2 Short Answers
  • 1 Long Answer
  • Peer assessment rubric

9. Homework Assignments

  • ML application report
  • Mind map creation
  • Dataset exploration with Python

10. Extension Activities

  • Use Teachable Machine
  • ML career research presentation
  • Ethics reflection
  • Create a quiz for peers

11. Reflection

  • 5-minute feedback form
  • Exit ticket: “One thing I learned, one question I still have.”
Note: Multimedia tools like YouTube, Kahoot, and Teachable Machine boost concept clarity and real-world understanding.
