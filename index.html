Lesson Plan: Introduction to Machine Learning

1. Target Audience

BCA Students (High Graders)

2. Duration

2–3 hours (can be split into two sessions)

3. Overview

This lesson introduces students to the basics of machine learning, including key concepts, types of ML, workflows, and real-world applications. Interactive activities and assessments ensure engagement and understanding.

4. Content Outline

  • Hook: ML video + Mentimeter poll
  • Intro to AI/ML + Types
  • ML Workflow & Tools (Google Teachable Machine)
  • Applications and real-world examples

5. Extension Activities

  • Use Teachable Machine
  • ML career research presentation
  • Ethics reflection
  • Create a quiz for peers

6. Homework Assignments

  • ML application report
  • Mind map creation
  • Dataset exploration with Python

7. Reflection

  • 5-minute feedback form
  • Exit ticket: “One thing I learned, one question I still have.”
Note: Multimedia tools like YouTube, Kahoot, and Teachable Machine boost concept clarity and real-world understanding.
